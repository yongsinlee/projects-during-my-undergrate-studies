
# 🚀 사용 방법 (Usage Guide)
'항소개팅'을 실행하고 서비스를 이용하는 단계별 가이드입니다.

## 1. 프로그램 실행 (Initial Setup)
MATLAB을 실행 후 example.fig(또는 example.m) 실행

##2. 회원가입 및 로그인 (Sign-up & Login)
회원가입:

메인 화면의 '회원가입' 버튼을 클릭합니다.

이름, 학번, 학과, 학년, 성별 등 필수 정보를 입력합니다.

입력된 정보는 로컬 데이터베이스(user_db.txt 등)에 안전하게 저장됩니다.

로그인:

생성한 학번과 정보를 바탕으로 로그인을 진행하여 메인 메뉴로 진입합니다.

## 3. GPS 데이터 업로드 및 시간표 생성 (Timetable Update)
가장 핵심적인 단계로, 본인의 GPS 로그를 기반으로 시간표를 구축합니다.

Timetable 창 진입: 메뉴에서 시간표(Timetable) 관련 버튼을 클릭하여 이동합니다.

CSV 파일 업로드:

upload 폴더에 미리 준비한 학번_요일.csv 파일을 넣습니다.

창 내의 '파일 불러오기' 또는 '업로드' 버튼을 클릭하여 해당 CSV 파일을 선택합니다.

시간표 업데이트 (Update):

'시간표 업데이트' 버튼을 클릭하면 MATLAB 알고리즘이 GPS 좌표를 분석합니다.

수집된 위도/경도 데이터를 세종대학교 건물 맵핑 데이터와 대조하여 사용자가 머문 건물을 판별합니다.

분석이 완료되면 GUI 상의 시간표 셀에 자동으로 수업 정보가 채워집니다.

##4. 상대방 매칭 (Matching)
생성된 시간표를 확인한 후, 상단 탭에서 매칭 목적(밥 친구, 공부 친구, 일반 친구)을 선택합니다.

'Match' 버튼을 누르면 현재 데이터베이스에 등록된 다른 사용자들의 패턴과 비교하여 최적의 상대를 화면에 표시해 줍니다.
