# 🚀 Separable Reconnaissance Glider Design & Fabrication
**무인항공기설계2: 분리형 정찰 글라이더 제작 프로젝트**

공기역학적 유동 간섭 원리를 이용해 별도의 기계적 장치 없이 공중에서 자연스럽게 분리되는 정찰용 글라이더를 설계하고 제작한 프로젝트입니다.

---

## 1. 📋 프로젝트 개요 (Project Overview)
* **미션**: 단일 발사 후 공중 분리 → 좌우 개별 활공 → 광범위 정찰 수행.
* **핵심 컨셉**: 복엽기 배치 시 발생하는 상/하 날개 간의 유동 간섭(양력/항력 변화)을 역이용하여 자연스러운 분리 유도.
* **설계 프로세스**: XFLR5를 이용한 유동 해석, CATIA를 활용한 기체 및 연결부 설계, 3D 프린팅(PLA+)을 통한 정밀 부품 제작.

## 2. 🛠️ 기체 제원 (Design Specification)
| 항목 | 사양 및 특징 |
| :--- | :--- |
| **에어포일 (Airfoil)** | NACA 4415 (저속 고양력 및 높은 양항비 확보) |
| **날개 치수 (Wing)** | Span 1m / Chord 0.11m / Aspect Ratio 10 |
| **주요 재료** | 카본 파이프(동체 지지대) + 3D 프린팅 리브 + 종이 필름 커버 |
| **분리 메커니즘** | 갈고리(Hook) 및 홈(Slot) 형태의 물리적 연결 장치 |



## 3. 📉 성능 해석 (Analysis & Performance)
* **양력 계수**: XFLR5 해석 결과, 받음각 0도 부근에서 약 0.35의 $C_L$ 값 도출.
* **양항비 (L/D)**: Wetted Aspect Ratio 계산을 통해 약 10 내외의 최대 양항비 예측.
* **활공 성능**: 고도 3m 발사 시 단일 기체는 약 20초, 분리형은 약 10초의 이론적 활공 시간 계산.

## 4. 🚀 비행 시험 및 최적화 (Flight Test & Optimization)
* **안정성 개선**: 초기 시험에서 분리 직후 상부 기체의 급격한 피치 상승 문제 발견 → 수평꼬리날개 스팬을 40cm로 확장하여 복원력 강화.
* **무게중심 최적화**: 주날개 중심 기준 4cm 뒤로 무게중심(CG)을 조정하여 비행 안정성 확보.
* **방향 제어**: 러더(Rudder) 각도를 7도로 설정하여 분리 후 기체가 좌우로 적절히 벌어지도록 유도.

## 5. 🏆 최종 결과 (Final Results)
* **비행 기록**: 초기 고도 2.5m 발사 시 비행 시간 2.8초, 비행 거리 18.1m 달성.
* **미션 성과**: 공중 분리 메커니즘 정상 작동 확인 및 좌우 이격 거리 6.1m 기록.
* **의의**: 항공 컨테이너 투하 및 무인기 긴급 탈출 시스템 기술로의 응용 가능성 확인.
