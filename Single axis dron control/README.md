# 아두파일럿(ArduPilot) 기반 드론 1축 자세 제어 프로젝트
본 프로젝트는 오픈소스 오토파일럿 플랫폼인 아두파일럿(ArduPilot) 라이브러리를 활용하여 드론의 핵심 제어 알고리즘을 직접 프로그래밍하고, 1자유도(1-DOF) 실험 장치를 통해 비행 제어 원리를 검증한 프로젝트입니다.

## 1. 프로젝트 개요

해당 과목: 메카트로닉스종합설계

목적: 비행 제어기(FC)의 동작 원리 이해 및 PID 제어 알고리즘 구현

주요 도구: APM 2.8(FC), MPU6050(IMU 센서), 아두이노 IDE, 아두파일럿 라이브러리

핵심 내용: 센서 데이터 추출, 조종기 신호 입력 처리, 모터 출력 제어 및 PID 튜닝

## 2. 주요 구현 특징
① 다중루프(Multi-loop) PID 제어기 설계
단순한 각도 제어를 넘어, 시스템의 안정성을 높이기 위해 **각도(Outer loop)**와 **각속도(Inner loop)**를 동시에 제어하는 다중루프 구조를 적용했습니다.

각도 제어: 목표 각도와 실제 각도의 오차를 바탕으로 목표 각속도를 계산합니다.

각속도 제어: 자이로 센서에서 얻은 실제 각속도를 피드백 받아 모터의 최종 출력을 결정하며, 시스템에 감쇠(Damping) 효과를 주어 진동을 억제합니다.

② 센서 데이터 처리 및 보정 (Calibration)
IMU 센서 정밀화: 가속도계와 자이로스코프의 오프셋(초기 오차)을 제거하여 정지 상태에서 정확한 0도를 유지하도록 보정했습니다.

자세 추정: MPU6050의 내부 프로세서(DMP)를 활용하여 노이즈가 적은 안정적인 오일러 각(Euler Angle) 데이터를 확보했습니다.

③ 하드웨어 인터페이스 및 통신
PPM 신호 분석: 조종기 수신기의 PPM 신호를 해석하여 다채널 입력을 효율적으로 처리했습니다.

전력 및 구동 제어: ESC(전자 변속기)와 BLDC 모터 간의 범위를 일치시키는 보정 작업을 통해 조종기 스틱 조작에 따른 선형적인 추력 발생을 구현했습니다.

## 3. 실험 결과 및 배운 점
게인 튜닝(Gain Tuning): 각도 P 게인과 각속도 P 게인을 단계별로 수정하며, 외란(외부 충격) 발생 시에도 신속하게 수평 상태로 복원되는 것을 확인했습니다.

데이터 분석: 시리얼 모니터로 출력된 목표값과 실제값의 데이터를 엑셀로 시각화하여 제어 성능을 객관적으로 분석했습니다.

성과: 이론으로만 접하던 PID 제어가 실제 하드웨어의 관성과 마찰 속에서 어떻게 작동하는지 깊이 있게 이해할 수 있었습니다.
